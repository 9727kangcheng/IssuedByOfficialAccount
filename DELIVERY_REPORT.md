## 🚀 工作流程

```
用户提供地点 + 素材
        ↓
技能自动处理：
  1. 提取视频封面（或用指定图片）
  2. 生成文案和标题
  3. 上传到公众号素材库
  4. 创建图文草稿
  5. 推送到草稿箱
        ↓
用户在公众号后台：
  1. 进入草稿箱
  2. 预览检查排版
  3. 发布到公众号
```

## ⚠️ 重要配置

**首次使用前必须配置 IP 白名单：**

1. 登录微信公众号后台
2. 进入 **开发** → **基本配置**
3. 找到 **IP 白名单** 设置
4. 添加当前 IP 地址

（首次上传时会显示当前 IP，复制添加即可）
