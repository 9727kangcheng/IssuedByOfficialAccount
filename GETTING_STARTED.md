## ⏱️ 5 分钟快速开始

### 第 1 步：了解工作流程（1 分钟）

```
你提供地点 + 素材
        ↓
技能自动生成文案、标题、排版
        ↓
上传到公众号、创建草稿、推送到草稿箱
        ↓
你在公众号后台预览 → 发布
```

### 第 2 步：配置 IP 白名单（3 分钟）

**首次使用前必须做这一步：**

1. 登录微信公众号后台
2. 进入 **开发** → **基本配置**
3. 找到 **IP 白名单** 设置
4. 添加当前 IP 地址

（首次上传时会显示当前 IP，复制添加即可）

### 第 3 步：准备素材（1 分钟）

准备：
- **地点名称** — 比如"京西翡翠湖"
- **图片或视频** — 本地文件路径
